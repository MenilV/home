# Windows Research - Pofalići House

## Overview

**Current Need:** Replace all windows in ~200m² townhouse
**Preference:** PVC or Aluminum (not wood)
**Goal:** Energy-efficient, good insulation

## Recommended Specifications

### For Sarajevo Climate (-20°C winter)

**Must Have:**
- [ ] Uw value ≤ 1.0 W/m²K (better = more efficient)
- [ ] Triple glazing (3 layers of glass)
- [ ] Low-E coating on glass
- [ ] Argon gas between glass layers
- [ ] 5+ chamber profile
- [ ] Proper installation with tape/foam

**Recommended Brands in BiH:**
- [ ] **Kommerling** - German quality, popular in BiH
- [ ] **Salamander** - Premium German
- [ ] **Deccco** - Good value, local
- [ ] **Wertbau** - German engineering

---

## Questions for Window Suppliers

1. What is the Uw value of the windows?
2. What glass package is included?
3. What is the lead time for order?
4. Do you include installation?
5. What warranty do you provide?
6. Can I see references?
